# AI Input 001

A comprehensive AI chat interface designed for modern LLM applications.

- **Model Switching** - Easily toggle between different AI models like GPT-4, Claude, and Gemini.
- **Smart Modes** - Integrated toggles for Search and DeepMind capabilities.
- **Attachment Menu** - Quick access for uploading images, documents, and connecting apps.
- **Adaptive Input** - Textarea that expands naturally with content.
- **Refined Animations** - Powered by Framer Motion for a premium, tactile feel.
