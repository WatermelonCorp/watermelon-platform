# AI Input 005

A specialized input interface designed for "Deep Thinking" AI models, featuring a toggleable mode and sophisticated status animations.

- **Dual Thinking Modes** - Toggle between "Normal" and "Deep Thinking" modes, changing the context of the AI's response.
- **Animated Status Effects** - Shimmering text and clip-path animations indicate the deep thinking state.
- **Premium Dark Aesthetic** - A deep black background with subtle blue gradients and blur effects for a futuristic feel.
- **Controlled State** - fully controlled component architecture for seamless integration with external state management.
