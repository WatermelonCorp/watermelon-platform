# AI Input 002

A refined and lightweight AI chat interface focusing on smooth interactions and minimal design.

- **Clean Chat Interface** - A sleek, minimal chat UI with a centrally positioned, sticky input bar.
- **Spring Animations** - Smooth message entry and status transitions powered by Framer Motion.
- **Async Responses** - Integrated support for handling asynchronous AI response simulated flows.
- **File Attachments** - Easily accessible plus button for future file upload integrations.
- **Interactive Send Button** - Dynamic arrow icon that rotates and reacts to input content.
- **Loading Indicators** - Subtle, animated typing indicators to signal background AI processing.
