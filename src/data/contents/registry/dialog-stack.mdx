## Introduction

A sophisticated dialog system that handles sequential flows with a beautiful stacked card aesthetic. Perfect for onboarding, invite flows, or multi-step processes where maintaining context is key.

## Features

- **Stacked Card UI**: Previous steps are visually preserved as layers behind the active card.
- **Dynamic Content Types**: Support for forms, walkthrough steps, and custom content.
- **Spring Animations**: Physics-based transitions for depth and scale changes using `motion/react`.
- **Integrated Navigation**: Fluid forward/back transitions within the stack.
- **Tailwind Dark Mode**: First-class support for system themes using `dark:` utility classes.
- **Premium Aesthetics**: Subtle blurs, elegant borders, and clean typography.
