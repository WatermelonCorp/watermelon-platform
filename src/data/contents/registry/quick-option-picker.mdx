# Quick Option Picker

A high-performance, visually stunning option picker designed for modern interfaces. It features:

- **Smooth Roll Animations**: Uses Framer Motion for fluid transitions between options.
- **Micro-interactions**: Subtle scale and hover effects for a premium user experience.
- **Dark Mode Support**: Fully compatible with both light and dark themes.
- **Customizable**: Easily extend with custom icons and labels.
