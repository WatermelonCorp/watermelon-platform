# Morphing Sidebar Controls

A high-fidelity configuration surface that morphs between layout and motion controls. Built for animation tooling, design inspectors, and power-user dashboards that demand deep control with elegant progressive disclosure.

- **Dual-Mode Interface**: Seamlessly switches between configuration and motion panels.
- **Progressive Feature Reveal**: Advanced controls appear contextually.
- **Spring Layout Animations**: Smooth transitions powered by `motion/react`.
- **Preset Ease Gallery**: Visual easing selector with interactive feedback.
- **Theme Adaptive**: Carefully tuned for light and dark environments.
