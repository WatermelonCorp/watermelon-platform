# Task Widget Disclosure

An interactive task widget designed for detailed project tracking. It features smooth disclosure transitions between a compact overview and a comprehensive view of subtasks, priority, and assignees.

## Features

- **Disclosure Animation**: Seamlessly expands from a compact badge to a full-featured widget.
- **Progress Tracking**: Real-time progress visualization with animated loading states.
- **Subtask Checklist**: Intuitive subtask list with distinct completion states.
- **Responsive Theme**: Built-in support for Tailwind CSS `dark:` utilities for consistent across light and dark themes.
- **Premium Aesthetics**: High-quality icons, curated color palettes, and spring-based animations for a tactile feel.
- **Interactive States**: Smooth hover and tap effects using `motion/react`.
