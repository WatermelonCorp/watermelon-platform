# Feedback Component

A premium feedback component designed for high-end user experiences. It features morphing icons that transition smoothly between collapsed and expanded states, providing intuitive visual feedback.

## Features
- **Adaptive Layout**: Transitions gracefully between a simple rating trigger and a detailed feedback form.
- **Mobile Responsive**: Optimized for all screen sizes, ensuring the feedback form fits perfectly on mobile devices while maintaining a premium desktop experience.
- **Premium Aesthetic**: Curated color palettes for both light and dark modes, ensuring a high-quality look and feel.
- **Micro-interactions**: Uses `framer-motion` for spring-based animations and layout transitions (morphing).
- **Interactive Feedback**: Includes loading states and submission confirmations for a complete user flow.
- **Tailwind Dark Mode**: Fully integrated with Tailwind CSS `dark:` utilities for seamless theme switching.
