# Feature Tour

A high-fidelity onboarding component designed to guide users through key features with elegance and clarity. It combines cinematographic transitions with tactile interactive elements to create a memorable first impression.

- **Cinematic Transitions**: Leverages blur, scale, and spring animations via Framer Motion for a fluid, high-end navigation experience.
- **Interactive Shining Titles**: Features a dynamic linear-gradient animation on headings, adding a premium, reflective sheen that captures attention.
- **Keyboard Navigation**: Fully accessible with built-in support for Arrow keys and Escape, ensuring a seamless experience for power users.
- **Custom Icon Support**: Each tour step supports unique React components as icons, allowing for highly visual and branded storytelling.
- **Tactile Indicators**: Smooth pagination dots provide clear progress tracking with elegant scaling and focus states.
