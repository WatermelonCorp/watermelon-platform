# Carousel Navigator

A premium carousel navigation system that responds to state changes with fluid color transitions and layout animations.

- **Theme-Adaptive Transitions**: The background color smoothly animates between themes as you navigate.
- **Progress Indicators**: Active dots expand and show a timed progress fill synced with the auto-play delay.
- **Spring-Driven Layout**: Uses Framer Motion's layout animations for smooth dot transitions.
- **Accessible Design**: Fully keyboard-navigable with clear focus states and aria labels.
