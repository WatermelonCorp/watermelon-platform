# Schedule Button

A premium, interactive button component designed for scheduling content. It features a sleek interface for date and time selection with smooth Framer Motion animations.

- **Interactive State Management**: Effortlessly toggle between a simple "Post" state and a detailed "Schedule" state with smooth layout transitions.
- **Date & Time Selection**: Integrated placeholders for date and time selection with customizable dropdown icons.
- **Micro-Animations**: Uses Framer Motion for spring-based animations, layout transitions, and hover/tap effects, providing a premium feel.
- **Theme-Adaptive Visuals**: Automatically adapts colors, borders, and shadows for both light and dark modes, ensuring consistency across themes.
- **Highly Customizable**: Easily swap out icons for the calendar, dropdown, and cancel actions to match your app's design language.
