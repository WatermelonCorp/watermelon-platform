# Issue Tracking

Features:

- Queue-based issue management with tab switching
- Interactive status, label, priority, and effort pickers
- Collapsible work items panel with card detail view
- Real-time discussion thread with AI-generated sub-tasks
- Add work item dialog with priority selection
- Attachment popover with upload, link, and docs options
