# Project Management Dashboard

Features:

- Comprehensive Settings layout allowing navigation across General, Workspace, Privacy, Notifications, Team Members, Billing, and Integrations tabs.
- Detailed Workspace settings including logo upload, workspace information, preferences (language, timezone).
- Integrations interface featuring connection statuses, management overlays, and one-click sync toggles.
- Sidebar navigation with collapsible sections.
