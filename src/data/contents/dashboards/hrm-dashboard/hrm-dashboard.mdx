# HRM Dashboard

Features:

- Comprehensive overview of total employees, attendance, and project counts
- Interactive attendance overview chart with 7-day, 30-day, and yearly views
- Visual breakdowns for working formats (On Site, Hybrid, Remote)
- Job overview tracking for active positions and interviews
- Responsive employee activity management with list and board view variants
- Dual-panel sidebar with an icon rail and secondary navigation
- Theme switching support and mobile-responsive layout
