# Meetings Dashboard

Features:

- Kanban board layout for categorizing meetings by status (Upcoming, Scheduled, Confirmed, Cancelled)
- Comprehensive filtering sidebar for meeting properties, time, location, and host
- Multi-platform integration support (Google Meet, Zoom, Slack, Discord)
- Quick actions for scheduling and importing meetings
- Detailed meeting cards with host info, duration, and related entity links
- Responsive design with sidebar drawers for mobile users
