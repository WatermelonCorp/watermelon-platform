# Mail Dashboard

A comprehensive dashboard design built to mimic the look and feel of a premium email client.

Features:

- Collapsible global sidebar for navigating different sections of the email client (Inbox, Drafts, Sent, etc.).
- Three-pane responsive layout: Global navigation sidebar, mail list view, and mail detail viewer.
- Mail List: Interactive list of emails supporting selection highlighting.
- Mail Viewer: Displays the full email content, sender details, and quick action buttons.
