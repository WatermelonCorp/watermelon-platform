# ERP

Features:

- Overview of total revenue, subscriptions, sales, and active users
- Date range picker for customized reporting
- Tabbed interface for Overview, Analytics, Reports, and Notifications
- Recent sales list with avatars and transaction details
- Bar chart for monthly revenue visualization
- User-friendly sidebar navigation with project switching
