# E-Commerce Dashboard

A fully featured e-commerce analytics dashboard with:

- **Metrics Cards** - Total orders, sales, visits, and conversion rate with trend indicators
- **Revenue Chart** - Area chart showing profit vs loss over time
- **Transactions Table** - Recent transaction list with status badges
- **Traffic Channel** - Pie chart showing traffic sources
